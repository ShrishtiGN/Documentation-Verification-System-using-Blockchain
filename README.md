# Documentation Verification System using Blockchain

## Overview
This project implements a Documentation Verification System utilizing blockchain technology. This system aims to ensure the integrity, authenticity, and non-repudiation of documents.

## Features
- Secure document verification using blockchain.
- Easy access to document status.
- User-friendly interface for submitting and verifying documents.

## Prerequisites
Before running the application, make sure you have the following installed:
- Node.js (version 14 or higher)
- npm (Node Package Manager)
- MongoDB (for local development)

## Setup Instructions
1. **Clone the Repository**  
   `git clone https://github.com/ShrishtiGN/Documentation-Verification-System-using-Blockchain.git`

2. **Navigate to the Project Directory**  
   `cd Documentation-Verification-System-using-Blockchain`

3. **Install the Dependencies**  
   `npm install`

4. **Setup the Database**  
   - Ensure MongoDB is running locally or connect to a cloud database.
   - Update the database connection string in the configuration file.

5. **Run the Application**  
   `npm start`

The application should now be running on `http://localhost:3000`.

## Usage Guide
- **Submit a Document:**  
   Navigate to the submit page and fill in the required details, upload the document, and click on submit.

- **Verify a Document:**  
   To verify a document, enter its unique ID on the verification page and click verify.
